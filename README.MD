# Conversational PDF RAG Chatbot

A Retrieval-Augmented Generation (RAG) application built with:

- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq LLM

## Features

- Multi-PDF Upload
- Conversational Q&A
- Chat History Aware Retrieval
- Groq-powered Responses

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
